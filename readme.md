大小脑协同的移臂小车精准建图与语义规划技术与系统实现

这里是其中的大脑部分。

- 设计理念如下：
离线地图层：建立3D的地图，并建立L2语义层
L2语义层包含物品的RGB，物品的xyz，物品的长宽高（如有）

大脑规划层：
1.人提出一个任务，大脑根据prompt将其划分为一系列子任务。
2.子任务的物体通过parse_obj_name再调用一次大模型从L2层的一部分objects字典中提取出所有的物品，并从中判断哪一个是我们的任务所需的目标
3.通过exec执行一个个子任务，在每个原子任务结束后，调用judgeLLM来判断原子任务是否成功，如果失败，就进行微调。如果还失败，就交给人判断。

小脑（非本项目负责，可以mock）：
1.机器人运动的底层api
2.L3点云避障建图
3.L2语义建图，通过YOLO和RGBD深度图进行质心计算，并输出L2层地图
